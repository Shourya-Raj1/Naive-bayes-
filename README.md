# 🧠 Naive Bayes Learning System

An interactive web-based machine learning application built using Streamlit that demonstrates the complete workflow of a Naive Bayes model.

---

## 🚀 Features

* 📁 Upload CSV dataset
* 🧹 Data preprocessing (cleaning & encoding)
* 📊 Exploratory Data Analysis (EDA)
* 🤖 Train Naive Bayes model
* 🔮 Make predictions
* 📈 Evaluate model performance

---

## 🛠️ Tech Stack

* Python
* Streamlit
* Pandas, NumPy
* Scikit-learn
* Matplotlib, Seaborn

---

## ⚙️ How to Run

```bash
git clone https://github.com/your-username/naive-bayes.git
cd naive-bayes
pip install -r requirements.txt
streamlit run app.py
```

---

## 📌 Project Workflow

Upload → Preprocess → EDA → Train → Predict → Evaluate

---

## 📊 Algorithm Used

Naive Bayes Classifier based on Bayes Theorem.

---

## 🎯 Purpose

This project is designed for students and beginners to understand machine learning concepts through an interactive interface.
